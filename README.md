This project is a web-based space exploration application designed to help users discover and interact with space-related media content. The main purpose of this project is to build an interactive platform that demonstrates API integration, dynamic data handling, and user-friendly design using frontend technologies.

The application allows users to search for space topics such as planets, galaxies, and space missions. Based on the search input, it fetches real-time data and displays images, videos, and audio in a responsive gallery format. Users can filter results by media type, sort them alphabetically, and explore content using a random discovery feature.

In addition to search functionality, the application displays a daily featured space image along with its title, date, and description. Users can also save their favorite items, which are stored locally in the browser and can be accessed later in a separate saved collection section.

This project uses the following public APIs:

* NASA Image and Video Library API
  Endpoint: [https://images-api.nasa.gov/search](https://images-api.nasa.gov/search)
  Used to fetch space-related images, videos, and audio based on user queries

* Astronomy Picture of the Day (APOD) API
  Endpoint: [https://api.nasa.gov/planetary/apod](https://api.nasa.gov/planetary/apod)
  Used to display a daily featured space image with details

The application is built using HTML, CSS, and JavaScript, focusing on API integration, responsive design, local storage, and interactive user experience.
